01

SSH One-Liner

Works on any machine with a terminal. No download, no signup, no binary. Every OS since 2005 has SSH built in. Just run one command and you're live.

ssh -p 443 -R0:localhost:3000 TOKEN@tunl.zyppr.xyz
Zero download HTTP & TCP Auto-HTTPS
02

CLI Binary (QUIC)

Download the zyppr binary for the fastest possible experience. QUIC transport (UDP-based, faster than TCP), auto-reconnect, config-as-code, and P2P mesh.

zyppr tunnel 3000 8080 --udp 25565
HTTP/TCP/UDP P2P Mesh Config-as-Code
03

Browser Tunnel

Open your browser, paste your token, and your localhost is live. Supports HTTP, streaming responses, and SSE. Nothing to install. Nothing to download. Nothing.

Open browser → Paste token → Tunnel is live
Zero install SSE Streaming Auto-reconnect
04

UDP Bridge (Zero-Download)

Tunnel UDP traffic — game servers, DNS, VoIP, IoT — with a single command. No binary to download. No tunneling service on Earth offers zero-download UDP.

node -e "$(curl -sL tunl.zyppr.xyz/bridge.js)" 25565 TOKEN

02 // 29 BUILT-IN FEATURES

Every Feature You Need. Built In.

Not plugins. Not add-ons. Not "coming soon." Every feature below is production-ready and included in your plan. 29 features. One platform.

Security & Protection

E2E Encryption

Every tunnel is encrypted end-to-end with verification. Each session gets a unique emoji fingerprint (🚀🌊💎🔑) so you can visually confirm no one is intercepting your traffic. Not just HTTPS — cryptographic proof.

AI Gateway WORLD-FIRST

Real-time threat detection on your tunnel. Blocks SQL injection, XSS, path traversal, and suspicious user agents automatically. No WAF subscription needed.

Rate Limiter

Set per-session request limits with burst tolerance. Protect your dev environment from runaway scripts or DDoS during demos. Configure via dashboard or API.

Developer Tools

Request Inspector

See every request hitting your tunnel — method, path, headers, body, response code, timing. Full visibility without a proxy tool. Captures everything in real-time.

Request Replay

Captured something interesting? Replay it with one click. Modify headers, body, or method before replaying. Full replay history with diff comparison.

Snapshots

Capture a point-in-time snapshot of all tunnel traffic. Persisted to disk — survives server restarts. Share snapshots with your team for debugging.

Auto-Swagger

Zyppr observes your live traffic and auto-generates OpenAPI documentation. Every path, method, and response code — documented by usage, not by hand.

DB Studio WORLD-FIRST

A built-in web SQL browser for your tunneled databases. Browse schemas, explore tables, and run read-only queries — all from your dashboard. Like phpMyAdmin, but inside your tunnel. Supports PostgreSQL, MySQL, and SQLite.

3 Databases
SQL Read-Only

Webhook Engineering

Webhook Queue

Capture every incoming webhook from Stripe, GitHub, Twilio, or any service. Queue them for later inspection. Never miss a delivery again.

Webhook Replay

Failed webhook? Re-deliver it with one click. Modify the payload before replaying. Filter by method, path, or status code.

Webhook Breakpoints EXCLUSIVE

Pause an incoming request mid-flight. Inspect it. Modify it. Then release it to your server. Like a debugger breakpoint, but for HTTP. 60-second auto-release ensures no permanent blocks.

Header Rules

Add, remove, or override any HTTP response header. Set CORS headers, security policies, or custom metadata without touching your backend code.

Advanced Networking

Custom Domains + Auto-SSL

Bring your own domain. Point a CNAME record, and Zyppr automatically provisions a Let's Encrypt TLS certificate. HTTPS on your custom domain in minutes, not days. Auto-renewal included.

SSL Auto-Cert
∞ Renewals

Edge Functions

Attach lightweight route handlers to your tunnel. Redirect paths, block IPs, return mock responses, or enable maintenance mode — without touching your server.

Edge Mesh WORLD-FIRST

Link multiple developer laptops into a private mesh network. Laptop A can call backend.zyppr.local and it routes to Laptop B. Service discovery for dev teams.

Chaos Engineering EXCLUSIVE

Inject 500ms latency. Return 5% random 500 errors. Simulate packet loss. Built-in fault injection to test your app's resilience before your users do.

P2P Mesh WORLD-FIRST

Connect two laptops directly via UDP hole punching — bypassing the server entirely. The CLI registers with the signaling server, exchanges NAT info, and punches through firewalls. If direct fails, falls back to server relay automatically. Near-zero latency for P2P data transfer.

P2P Direct UDP
~0ms Added Latency

Operations & Management

Live Dashboard

Real-time traffic stats, QR code sharing, request logs with color-coded status codes. View everything from the SSH terminal or the web dashboard.

Tunnel Sharing

Share tunnel access with teammates via secure links. Read-only or write access. Set expiry times. Max 10 active shares per session for security.

Config-as-Code

Define all your tunnels in a zyppr.yml file. Run zyppr up and every tunnel starts simultaneously. Version-control your tunnel setup.

Health Checks

Automatic uptime monitoring with branded status pages. Know instantly if your localhost goes down. Configurable check intervals and notification thresholds.

Custom Subdomains

Reserve your own cosmic identity. myapp.tunl.zyppr.xyz instead of random strings. Persistent across sessions on Pro plans.

KV Store

Session-scoped key-value storage. Store flags, feature toggles, or configuration that your edge functions can access. Lightweight and ephemeral.
