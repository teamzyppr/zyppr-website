Dual Architecture

Zero-Download +
Full CLI Power

Zyppr follows a dual architecture. Use the zero-download SSH client for instant tunneling with no install — or use the Zyppr CLI binary for advanced features like QUIC transport, UDP tunneling, P2P mesh networking, and config-as-code. Both paths, one platform.

Zero Download (SSH) CLI Binary (QUIC) World's First UDP
comparison
# Path 1: Zero Download (SSH)
$ ssh -p 443 -R0:localhost:3000 qr@free.zyppr.online
✓ Works instantly. No install needed.
# Path 2: CLI Binary (advanced)
$ zyppr tunnel 3000 --udp 25565
✓ QUIC, UDP, P2P mesh, config-as-code.
