REST API

API Reference v2.0

Complete REST API for Zyppr Tunnels. All endpoints return JSON. Authentication is required for most endpoints.

Authentication
Include your token as a query parameter: ?token=YOUR_TOKEN
Or as a header: Authorization: Bearer YOUR_TOKEN

Session Management

GET /api/v1/status

Get current server status and version info.

GET /api/v1/tunnels

List all active tunnels for the authenticated user. Returns session ID, URL, protocol, port, uptime, and traffic stats.

POST /api/v1/kill

Kill an active tunnel session. Body: { "session_id": "..." }

Inspector & Replay

GET /api/v1/inspect

Get captured requests for a session. Returns method, path, headers, body, status code, and timing for each request.

POST /api/v1/replay

Replay a captured request, optionally with modifications. Body: { "method": "POST", "path": "/api/test", "headers": {}, "body": "..." }

GET /api/v1/replay/history

Get replay history for a session — all replayed requests with original and modified versions.

Webhooks

POST /api/v1/webhooks/toggle

Enable or disable webhook queuing. Body: { "enabled": true }

GET /api/v1/webhooks

Get queued webhooks. Returns method, path, headers, body, and timestamp for each queued request.

POST /api/v1/webhooks/replay

Replay a queued webhook delivery. Body: { "id": "wh-abc123" }

POST /api/v1/webhooks/breakpoints

Enable/disable webhook breakpoints. When enabled, incoming requests are paused for inspection (auto-releases after 60s). Body: { "enabled": true }

Security

POST /api/v1/rate-limit

Configure rate limiting. Body: { "requests_per_minute": 60, "burst": 10 }

GET /api/v1/ai-gateway/stats

Get AI Gateway statistics — threat summary, blocked request count, traffic classification breakdown.

POST /api/v1/header-rules

Add, set, or remove response headers. Body: { "action": "set", "header": "X-Custom", "value": "hello" }

GET /api/v1/e2e/verify

Get E2E encryption verification info — emoji fingerprint and hex fingerprint for the session.

Snapshots

POST /api/v1/snapshot

Create a snapshot of tunnel traffic. Body: { "paths": ["/", "/api/users"] }

GET /api/v1/snapshots

List all snapshots for the session.

GET /api/v1/snapshots/:id/serve

Serve a specific snapshot page. Query: ?path=/api/users

DELETE /api/v1/snapshots/:id

Delete a snapshot.

Sharing

POST /api/v1/share/create

Create a share link. Body: { "access": "read|write", "ttl": "24h" }. Max 10 active shares per session.

POST /api/v1/share/revoke

Revoke a share link. Body: { "share_id": "abc123" }

GET /api/v1/shares

List all active shares for the session.

Edge Functions

POST /api/v1/edge/rules

Create an edge rule. Body: { "name": "...", "path": "/old", "action": "redirect|block|mock|modify", "target": "/new" }

GET /api/v1/edge/rules

List all edge rules for the session with hit counts.

DELETE /api/v1/edge/rules/:name

Remove an edge rule by name.

Edge Mesh

POST /api/v1/mesh/join

Join a mesh network. Body: { "mesh_id": "team-alpha", "hostname": "backend", "port": "3000", "protocol": "http" }

POST /api/v1/mesh/leave

Leave a mesh network. Body: { "mesh_id": "team-alpha" }

GET /api/v1/mesh/topology

Get full mesh topology — all members, hostnames, health status (online/offline/degraded), last-seen times.

GET /api/v1/mesh/resolve

Resolve a .zyppr.local hostname to a session. Query: ?hostname=backend.zyppr.local

GET /api/v1/mesh/list

List all meshes the session belongs to.

P2P Mesh

POST /api/p2p/register

Register a peer for P2P connections. Body: { "peer_id": "...", "public_addr": "ip:port" }

GET /api/p2p/peers/:peer_id

Look up a peer's public address for UDP hole punching.

POST /api/p2p/relay

Create a TURN-like relay for peers that can't establish direct connections.

Chaos Engineering

POST /api/v1/chaos

Configure chaos parameters. Body: { "latency_ms": 500, "error_rate": 0.05, "error_code": 500 }

GET /api/v1/chaos

Get current chaos configuration for the session.

DB Studio

POST /api/v1/db-studio/connect

Connect to a tunneled database. Body: { "driver": "postgres|mysql|sqlite", "dsn": "..." }

GET /api/v1/db-studio/schema

Get database schema — all tables, columns, and types.

GET /api/v1/db-studio/tables/:name

Browse rows in a specific table (paginated, read-only).

POST /api/v1/db-studio/query

Execute a read-only SQL query. Body: { "sql": "SELECT * FROM users LIMIT 10" }

KV Store & Health

GET /api/v1/kv

Get a value by key. Query: ?key=feature_flag

POST /api/v1/kv

Set a key-value pair. Body: { "key": "...", "value": "..." }

DELETE /api/v1/kv

Delete a key. Query: ?key=feature_flag

POST /api/v1/health/config

Configure health checks. Body: { "path": "/health", "interval": 60 }

GET /api/v1/health

Get health check status for the session — current state, last check time, consecutive failures.

Auto-Generated Docs

GET /api/v1/swagger

Get auto-generated OpenAPI 3.0 spec based on live traffic observation. Returns JSON with all observed paths, methods, and response codes.
